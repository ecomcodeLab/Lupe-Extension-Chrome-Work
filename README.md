# Web-Lupe Pro (Chrome Extension)

![Web-Lupe Pro Icon](icon128.png)

Die **Web-Lupe Pro** ist eine leistungsstarke Chrome-Erweiterung, die Ihnen eine hochpräzise digitale Lupe und einen Farb-Picker direkt in Ihrem Browser bietet. Vergrößern Sie beliebige Bereiche einer Webseite und lesen Sie den genauen Farbcode (Hex-Wert) eines Pixels aus – ideal für Webdesigner, Entwickler oder jeden, der Details auf Webseiten genau betrachten möchte.

---

## Funktionen

*   **Digitale Lupe:** Vergrößern Sie jeden Bereich einer Webseite, indem Sie die Maus darüber bewegen.
*   **Farb-Picker:** Lesen Sie den Hex-Farbcode des Pixels in der Mitte der Lupe aus und kopieren Sie ihn direkt in die Zwischenablage.
*   **Aktivierung per Klick:** Schalten Sie die Lupe einfach durch Klicken auf das Erweiterungs-Icon in Ihrer Chrome-Symbolleiste ein und aus.
*   **Konfigurierbare Shortcuts:**
    *   **Lupe An/Aus:** Standardmäßig `Alt+Z` (kann in den Einstellungen geändert werden).
    *   **Farbe kopieren:** Standardmäßig `Alt+C` (kann in den Einstellungen geändert werden).
*   **Lupe schließen:** Drücken Sie die `ESC`-Taste, um die Lupe schnell zu deaktivieren.
*   **Anpassbare Einstellungen:** Stellen Sie den Zoom-Faktor und die Größe der Lupe in den Erweiterungseinstellungen ein.
*   **Kein Mauszeiger:** Wenn die Lupe aktiv ist, wird der normale Mauszeiger ausgeblendet, um eine präzisere Ansicht zu ermöglichen.
*   **Zentrales Fadenkreuz:** Ein kleines Fadenkreuz in der Mitte der Lupe hilft Ihnen, den genauen Punkt für die Farbauswahl zu treffen.

## Installation (für unerfahrene Benutzer)

Da diese Erweiterung noch nicht im Chrome Web Store verfügbar ist, müssen Sie sie manuell installieren. Keine Sorge, das ist einfacher, als es klingt!

1.  **Laden Sie die Erweiterung herunter:**
    *   Klicken Sie auf diesen Link, um das Projekt von GitHub herunterzuladen: [Web-Lupe Pro auf GitHub](https://github.com/ecomcodeLab/Lupe-Extension-Chrome-Work)
    *   Klicken Sie auf der GitHub-Seite auf den grünen Button `<> Code` und wählen Sie dann `Download ZIP`.
    *   Speichern Sie die ZIP-Datei auf Ihrem Computer (z.B. im Ordner "Downloads").

2.  **Entpacken Sie die Datei:**
    *   Gehen Sie zu dem Ordner, in dem Sie die ZIP-Datei gespeichert haben.
    *   Klicken Sie mit der rechten Maustaste auf die heruntergeladene Datei (`Lupe-Extension-Chrome-Work-main.zip`).
    *   Wählen Sie `Alle extrahieren...` oder `Hier entpacken` (der genaue Wortlaut kann je nach Betriebssystem variieren).
    *   Es wird ein neuer Ordner mit dem Namen `Lupe-Extension-Chrome-Work-main` erstellt. Merken Sie sich, wo dieser Ordner ist!

3.  **Öffnen Sie Chrome und die Erweiterungsverwaltung:**
    *   Öffnen Sie Google Chrome.
    *   Geben Sie in die Adressleiste `chrome://extensions` ein und drücken Sie die `Enter`-Taste.
    *   Alternativ können Sie auf das **Drei-Punkte-Menü** (oben rechts) klicken, dann `Weitere Tools` > `Erweiterungen` auswählen.

4.  **Aktivieren Sie den Entwicklermodus:**
    *   Suchen Sie auf der Erweiterungsseite oben rechts den Schalter `Entwicklermodus` und schalten Sie ihn **ein**.

5.  **Laden Sie die entpackte Erweiterung:**
    *   Nachdem der Entwicklermodus aktiviert ist, sehen Sie oben links einen Button `Entpackte Erweiterung laden`. Klicken Sie darauf.
    *   Es öffnet sich ein Fenster, in dem Sie einen Ordner auswählen können. Navigieren Sie zu dem Ordner, den Sie in Schritt 2 entpackt haben (`Lupe-Extension-Chrome-Work-main`).
    *   Wählen Sie diesen Ordner aus und klicken Sie auf `Ordner auswählen`.

6.  **Fertig!**
    *   Die "Web-Lupe Pro" sollte nun in Ihrer Liste der Erweiterungen erscheinen.
    *   Sie können das Erweiterungs-Icon in Ihrer Chrome-Symbolleiste (oft neben der Adressleiste) finden. Falls es nicht sichtbar ist, klicken Sie auf das Puzzle-Symbol (Erweiterungen) und pinnen Sie die Web-Lupe Pro an.

## Nutzung

1.  **Lupe aktivieren/deaktivieren:**
    *   Klicken Sie auf das **Lupe-Icon** in Ihrer Chrome-Symbolleiste.
    *   Oder drücken Sie den konfigurierten Shortcut (Standard: `Alt+Z`).
2.  **Farbe kopieren:**
    *   Wenn die Lupe aktiv ist, bewegen Sie die Maus über den gewünschten Pixel.
    *   Drücken Sie den konfigurierten Shortcut (Standard: `Alt+C`). Der Hex-Farbcode wird in Ihre Zwischenablage kopiert und eine kurze Bestätigung wird angezeigt.
3.  **Lupe schließen:**
    *   Drücken Sie die `ESC`-Taste.
4.  **Einstellungen ändern:**
    *   Klicken Sie mit der rechten Maustaste auf das Lupe-Icon in Ihrer Chrome-Symbolleiste und wählen Sie `Optionen`. Hier können Sie Zoom, Größe und Shortcuts anpassen.

---

## Wichtiger Hinweis

Die Web-Lupe Pro funktioniert **nicht** auf speziellen Chrome-Seiten wie `chrome://settings`, `chrome://extensions` oder `about:blank`, da Chrome aus Sicherheitsgründen den Zugriff auf diese Seiten für Erweiterungen blockiert.
